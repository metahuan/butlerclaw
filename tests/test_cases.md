## 测试概述

### 项目简介

OpenClaw-Butler 是一个跨平台的 OpenClaw 安装管理助手，提供：
- 一键安装 OpenClaw 及依赖
- 技能管理（安装/卸载/更新）
- 版本管理（检查/更新）
- 系统诊断与修复
- 配置备份与恢复

### 测试范围

| 模块 | 功能点 | 测试类型 |
|------|--------|----------|
| 安装管理 | 安装向导、模型配置 | 功能、异常、兼容性 |
| 技能管理 | 技能列表、安装/卸载/更新 | 功能、性能、异常 |
| 版本管理 | 版本检查、一键更新 | 功能、网络、异常 |
| 诊断工具 | 环境诊断、一键修复、报告导出 | 功能、安全、性能 |
| 设置 | 清除缓存、备份恢复 | 功能、数据安全 |

### 测试环境

| 环境项 | 配置 |
|--------|------|
| 操作系统 | Windows 10/11, macOS 12+, Ubuntu 20.04+ |
| Python | 3.10+ |
| Node.js | 18+ (被测对象依赖) |
| 分辨率 | 1920x1080, 1366x768 |
