# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial open source release preparation
- GitHub Actions CI/CD workflows
- Issue and PR templates
- Contributing guidelines
- Security policy

## [2.0.0] - 2026-03-18

### Added
- Cross-platform support (Windows, macOS, Linux)
- Skill management system with built-in skill catalog
- Version management and auto-update functionality
- Diagnostic tools with export capability
- Web-based UI components
- ClawHub integration for skill marketplace
- API key security management
- Multi-language support foundation

### Changed
- Migrated from single installer to full-featured assistant
- Improved UI with modern tkinter theming
- Enhanced error handling and user feedback
- Optimized PyInstaller build process

### Fixed
- Node.js detection on various platforms
- PATH configuration issues on Linux/macOS
- Character encoding issues in Windows console
- Permission issues during installation

## [1.0.0] - 2025-03-08

### Added
- Initial release
- Windows installer with GUI
- Automatic Node.js installation
- OpenClaw CLI installation
- Model configuration (domestic and international)
- tkinter-based user interface
