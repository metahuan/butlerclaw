# OpenClaw 安装助手 - 打包成桌面程序

使用 **PyInstaller** 将「OpenClaw 管家」打包为各平台可独立运行的桌面程序，用户无需安装 Python 即可使用。

## 环境要求

- Python 3.8+（已安装 tkinter）
- 各平台在**对应系统**上打包（Windows 上打 Windows 包，macOS 上打 macOS 包，Linux 上打 Linux 包）

## 安装打包依赖

```bash
pip install -r requirements-build.txt
```

或单独安装：`pip install pyinstaller`

## 各平台打包步骤

### Windows

1. 打开命令提示符或 PowerShell，进入本项目目录。
2. 双击运行 **`build_windows.bat`**，或在命令行执行：
   ```cmd
   build_windows.bat
   ```
3. 完成后在 **`dist\`** 目录下得到 **`OpenClaw安装助手.exe`**。
   - **单文件模式**：一个 exe，双击即可运行，无需 Python。
   - **无控制台窗口**：使用 `--windowed`，不会弹出黑色命令行窗口。

### macOS

1. 终端进入本项目目录。
2. 赋予执行权限并运行：
   ```bash
   chmod +x build_macos.sh
   ./build_macos.sh
   ```
3. 完成后在 **`dist/`** 下得到 **`OpenClaw安装助手.app`**。
   - 可拖到「应用程序」文件夹，或直接双击运行。

### Linux

1. 终端进入本项目目录。
2. 执行：
   ```bash
   chmod +x build_linux.sh
   ./build_linux.sh
   ```
3. 完成后在 **`dist/`** 下得到可执行文件 **`OpenClaw安装助手`**，运行：
   ```bash
   ./dist/OpenClaw安装助手
   ```

## 可选：自定义图标

- **Windows**：准备 `.ico` 文件，在 `build_windows.bat` 的 PyInstaller 命令中增加：
  ```text
  --icon=你的图标.ico
  ```
- **macOS**：可对生成的 `.app` 使用「显示简介」→ 拖入图标，或通过 `--icon=图标.icns` 指定（需先制作 .icns）。
- **Linux**：可加 `--icon=图标.png`（视桌面环境而定）。

## 单文件 vs 目录模式

- 当前脚本使用 **单文件**（Windows/Linux 的 `--onefile`）：一个 exe/可执行文件，首次运行略慢（自解压），便于分发。
- 若希望启动更快，可改为**目录模式**：去掉 `--onefile`，会生成一个文件夹，内含 exe 与依赖库，整体体积略大但启动更快。

## 分发给用户

- **Windows**：直接分发 `OpenClaw安装助手.exe`，用户双击运行即可；无需安装 Python。
- **macOS**：分发 `OpenClaw安装助手.app`，若从网络下载，用户可能需在「系统偏好设置 → 安全性与隐私」中允许打开。
- **Linux**：分发 `OpenClaw安装助手` 并告知 `chmod +x` 后运行，或打成 .deb/.rpm 等包再安装。

## 注意事项

- 助手会按当前系统自动调用对应平台的安装器（Windows → openclaw_installer_v2，macOS/Linux → 各自脚本），打包时只会包含当前平台的安装逻辑，体积更小。
- 若杀毒软件误报 exe，可加入白名单或使用目录模式 + 代码签名（需自行配置证书）。

## 最快打包方式（Windows）

如果 `build_windows.bat` 因环境问题不好用，可以直接在命令行里手工跑两条命令，最快打出 exe：

1. 在资源管理器里进入项目目录：

   ```text
   e:\公司包包\openclaw行程文件\openclaw安装程序
   ```

   在空白处按住 Shift + 右键 → 选择「在此处打开 PowerShell 窗口」（或命令提示符）。

2. 在打开的窗口里依次执行（优先试 `python`，不行就换 `py`）：

   ```bash
   python -m pip install -r requirements-build.txt
   python -m PyInstaller --noconfirm --onefile --windowed --name "ButlerClaw龙虾管家" openclaw_assistant.py
   ```

   如果 `python` 不是命令，就用：

   ```bash
   py -m pip install -r requirements-build.txt
   py -m PyInstaller --noconfirm --onefile --windowed --name "ButlerClaw龙虾管家" openclaw_assistant.py
   ```

3. 成功后，在同目录下的 `dist\` 里面会出现：

   ```text
   dist\OpenClaw安装助手.exe
   ```

   直接双击这个 exe 就可以运行。

## 修改程序后的重新打包

每次修改 `openclaw_assistant.py`、安装器或技能/诊断等功能后：

- **必须重新执行一次对应平台的打包脚本**（如 Windows 的 `build_windows.bat`），或者按上面的「最快打包方式」重新运行 PyInstaller，才能让新的功能被打进 exe / app 中。
