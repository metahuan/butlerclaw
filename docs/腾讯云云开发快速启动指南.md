# OpenClaw Hub - 腾讯云云开发快速启动指南

## 一、环境准备

### 1.1 注册腾讯云账号

1. 访问 https://cloud.tencent.com
2. 注册账号并完成实名认证
3. 开通云开发（CloudBase）服务

### 1.2 安装工具

```bash
# 安装 Node.js (v18+)
https://nodejs.org/

# 安装云开发 CLI
npm install -g @cloudbase/cli

# 验证安装
tcb --version
```

---

## 二、创建项目

### 2.1 初始化云开发环境

```bash
# 登录腾讯云
tcb login

# 创建环境（选择合适的地域）
tcb env:create openclaw-hub-dev    # 开发环境
tcb env:create openclaw-hub-prod   # 生产环境

# 查看环境列表
tcb env:list
```

### 2.2 初始化项目

```bash
# 创建项目目录
mkdir openclaw-hub
cd openclaw-hub

# 初始化云开发项目
tcb init

# 选择配置：
# - 环境：openclaw-hub-dev
# - 语言：Node.js
# - 模板：空模板
```

## 四、快速开始

### 4.1 配置云开发

创建 `cloudbaserc.json`:

```json
{
  "version": "2.0",
  "envId": "openclaw-hub-dev",
  "functionRoot": "functions",
  "functions": [
    {
      "name": "user-api",
      "timeout": 10,
      "memorySize": 256,
      "handler": "index.main"
    },
    {
      "name": "skill-api",
      "timeout": 10,
      "memorySize": 256,
      "handler": "index.main"
    },
    {
      "name": "pay-api",
      "timeout": 10,
      "memorySize": 256,
      "handler": "index.main"
    },
    {
      "name": "notify-api",
      "timeout": 10,
      "memorySize": 256,
      "handler": "index.main"
    }
  ],
  "hosting": {
    "public": "web/dist",
    "ignore": ["node_modules"]
  }
}
```

### 4.2 初始化数据库

```bash
# 进入数据库目录
cd database

# 运行初始化脚本
node init.js
```

### 4.3 部署云函数

```bash
# 安装依赖
cd functions/user-api
npm install
cd ../..

# 部署所有云函数
tcb fn deploy --all

# 或者单独部署
tcb fn deploy user-api
tcb fn deploy skill-api
```

### 4.4 部署前端

```bash
# 进入前端目录
cd web

# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 部署到云托管
cd ..
tcb hosting deploy web/dist
```

### 5.2 前端页面模板

```javascript
// web/src/pages/template.js
import { useState, useEffect } from 'react';
import { api } from '../utils/api';

export default function TemplatePage() {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    fetchData();
  }, []);

  async function fetchData() {
    try {
      setLoading(true);
      const result = await api.callFunction({
        name: 'template-api',
        data: {
          action: 'example',
          data: { test: true }
        }
      });
      
      if (result.result.code === 200) {
        setData(result.result.data);
      } else {
        setError(result.result.message);
      }
    } catch (err) {
      setError(err.message);
    } finally {
      setLoading(false);
    }
  }

  if (loading) return <div>Loading...</div>;
  if (error) return <div>Error: {error}</div>;

  return (
    <div className="container mx-auto px-4 py-8">
      <h1 className="text-2xl font-bold mb-4">Template Page</h1>
      <pre className="bg-gray-100 p-4 rounded">
        {JSON.stringify(data, null, 2)}
      </pre>
    </div>
  );
}
```

### 5.3 API 工具封装

```javascript
// web/src/utils/tcb.js
import tcb from 'tcb-js-sdk';

// 初始化云开发
const app = tcb.init({
  env: 'your-env-id'  // 替换为你的环境ID
});

// 导出服务
export const auth = app.auth();
export const db = app.database();
export const storage = app.storage();

// 调用云函数
export async function callFunction(options) {
  try {
    const result = await app.callFunction(options);
    return result;
  } catch (error) {
    console.error('Cloud function error:', error);
    throw error;
  }
}

// 上传文件
export async function uploadFile(cloudPath, filePath) {
  try {
    const result = await storage.uploadFile({
      cloudPath,
      filePath
    });
    return result;
  } catch (error) {
    console.error('Upload error:', error);
    throw error;
  }
}

// 获取临时链接
export async function getTempFileURL(fileList) {
  try {
    const result = await storage.getTempFileURL({
      fileList
    });
    return result;
  } catch (error) {
    console.error('Get temp URL error:', error);
    throw error;
  }
}
```

```javascript

import { callFunction } from './tcb';

export const api = {
```

## 六、常用命令

### 6.1 云函数管理

```bash
# 部署云函数
tcb fn deploy <function-name>

# 部署所有云函数
tcb fn deploy --all

# 查看云函数列表
tcb fn list

# 查看云函数日志
tcb fn log <function-name>

# 删除云函数
tcb fn delete <function-name>
```

### 6.2 数据库管理

```bash
# 导出数据
tcb db export -c users -o users.json

# 导入数据
tcb db import -c users -f users.json

# 查看集合列表
tcb db collection:list
```

### 6.3 存储管理

```bash
# 上传文件
tcb storage upload local/path cloud/path

# 下载文件
tcb storage download cloud/path local/path

# 删除文件
tcb storage delete cloud/path

# 列出文件
tcb storage list cloud/path
```

### 6.4 托管管理

```bash
# 部署静态网站
tcb hosting deploy local/dir -e env-id

# 配置自定义域名
tcb hosting domain:create example.com -e env-id

# 查看托管信息
tcb hosting detail -e env-id
```

## 九、常见问题

### Q1: 云函数调用失败？

**检查项**：
- 云函数是否已部署
- 环境ID是否正确
- 入参格式是否正确
- 查看云函数日志

### Q2: 数据库查询慢？

**解决方案**：
- 创建索引
- 优化查询条件
- 使用分页
- 避免嵌套查询

### Q3: 如何迁移到生产环境？

**步骤**：
1. 创建生产环境
2. 部署云函数到生产环境
3. 导入生产数据
4. 配置生产域名
5. 更新前端环境配置

### Q4: 如何备份数据？

**方法**：
```bash
# 自动备份（控制台配置）
# 手动导出
tcb db export -c users -o backup/users-$(date +%Y%m%d).json
```
