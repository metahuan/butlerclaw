### 内容

```markdown
## 🎯 Butlerclaw - OpenClaw 生态管家

大家好！我想分享一个为 OpenClaw 深度用户打造的桌面管理工具。

### ✨ 核心功能

#### 1. 多实例管理
- 🔹 本地实例自动发现
- 🔹 SSH 远程连接管理
- 🔹 WebSocket 实时监控
- 🔹 统一命令执行接口

#### 2. 成本控制
- 📊 API 调用记录与统计
- 💰 按模型/时间段成本分析
- 🚨 预算设置与多级告警
- 📈 成本趋势可视化

#### 3. 团队协作
- 👥 团队创建与成员管理
- 🔐 RBAC 权限控制
- 🔄 配置同步与共享
- 📝 操作审计日志

#### 4. 现代化 UI
- 🎨 暗色/亮色/自动主题
- 📱 响应式布局
- 🌐 嵌入式 WebView 图表
- ⚡ 流畅的交互体验

### 🏗️ 架构亮点

```
┌─────────────────────────────────────┐
│         Butlerclaw v2.0             │
├─────────────────────────────────────┤
│  UI Layer: ttkbootstrap + WebView   │
│  App Layer: Event Bus (async)       │
│  Domain Layer: Instance/Cost/Team   │
│  Infra Layer: SQLite + JSONL        │
└─────────────────────────────────────┘
```
