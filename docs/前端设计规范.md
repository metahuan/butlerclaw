# OpenClaw Hub - 前端设计规范

## 一、设计原则

### 1.1 设计理念

- **简洁现代**: 干净的界面，充足的留白
- **用户友好**: 直观的导航，清晰的信息层级
- **响应式**: 完美适配桌面、平板、手机
- **高性能**: 快速加载，流畅交互

### 1.2 色彩系统

```css
/* 主色调 */
```

## 四、响应式设计

### 4.1 断点设置

```css
/* 移动端 */
@media (max-width: 640px) {
  /* 单列布局 */
  /* 隐藏部分元素 */
  /* 调整字体大小 */
}

/* 平板 */
@media (min-width: 641px) and (max-width: 1024px) {
  /* 两列布局 */
}

/* 桌面 */
@media (min-width: 1025px) {
  /* 多列布局 */
  /* 完整功能 */
}
```

### 4.2 响应式规则

| 元素 | 移动端 | 平板 | 桌面 |
|------|--------|------|------|
| 导航 | 汉堡菜单 | 简化导航 | 完整导航 |
| 技能网格 | 1列 | 2列 | 4列 |
| 字体大小 | 14px | 15px | 16px |
| 间距 | 16px | 24px | 32px |
| 侧边栏 | 隐藏 | 可折叠 | 固定显示 |
