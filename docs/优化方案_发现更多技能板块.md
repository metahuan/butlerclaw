# OpenClaw Butler - "发现更多技能"板块优化方案

## 当前现状分析

### 现有功能
1. **技能列表展示** - 显示内置技能（BUNDLED_SKILLS）和已安装技能
2. **搜索功能** - 按名称、ID、描述搜索
3. **分类筛选** - 按分类（工具、开发、效率、媒体、其他）筛选
4. **技能详情** - 显示技能信息、安装/卸载/更新操作
5. **安装队列** - 批量安装、进度追踪（新集成）

### 存在的问题

#### 1. 数据来源有限
- 仅显示内置技能（约20个）
- 无法从 ClawHub 获取最新技能
- 没有技能推荐算法

#### 2. 信息展示单一
- 仅显示名称、描述、图标
- 缺少下载量、评分、版本等元信息
- 没有技能预览或截图

#### 3. 交互体验待提升
- 没有批量选择功能
- 缺少排序选项（按下载量、评分、更新时间）
- 没有收藏/关注功能

#### 4. 发现机制不足
- 没有热门技能推荐
- 没有基于使用历史的推荐
- 没有新技能提醒

#### 1.3 本地缓存机制

```python
class SkillCache:
    """技能列表本地缓存"""
    
    def __init__(self, cache_dir: str = None):
        self.cache_file = os.path.join(cache_dir or "~/.openclaw", "skills_cache.json")
        self.cache_ttl = 3600  # 1小时
    
    def get(self) -> List[Dict]:
        """获取缓存的技能列表"""
        if not self._is_valid():
            return []
        
        with open(self.cache_file, 'r') as f:
            return json.load(f)
    
    def set(self, skills: List[Dict]):
        """更新缓存"""
        with open(self.cache_file, 'w') as f:
            json.dump({
                'timestamp': datetime.now().isoformat(),
                'skills': skills
            }, f)
    
    def _is_valid(self) -> bool:
        """检查缓存是否有效"""
        if not os.path.exists(self.cache_file):
            return False
        
        with open(self.cache_file, 'r') as f:
            data = json.load(f)
        
        cached_time = datetime.fromisoformat(data['timestamp'])
        return (datetime.now() - cached_time).seconds < self.cache_ttl
```
