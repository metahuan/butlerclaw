# OpenClaw Butler 运营网站 - 产品设计策划方案

## 一、项目概述

### 1.1 项目定位
**OpenClaw Hub** - OpenClaw 技能生态平台

一个连接用户、开发者和技能的综合性平台，提供：
- 用户：下载软件、发现技能、一键安装
- 开发者：上传技能、获取收益、社区交流
- 平台：运营生态、会员服务、数据分析

### 1.2 目标用户

| 用户类型 | 需求 | 占比 |
|---------|------|------|
| 普通用户 | 下载软件、发现实用技能 | 70% |
| 进阶用户 | 定制技能组合、会员服务 | 20% |
| 开发者 | 上传技能、获取收益 | 10% |

### 1.3 核心价值主张

**对用户**：
- 一键安装 OpenClaw 和技能
- 发现海量实用技能
- 个性化推荐

**对开发者**：
- 技能变现渠道
- 开发者工具支持
- 社区推广资源

**功能**：

```

   ↓
4. 设置价格（免费/付费/订阅）
```

### 4.2 数据库设计

**核心表结构**：

```sql
-- 用户表
CREATE TABLE users (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    avatar VARCHAR(255),
    role ENUM('user', 'developer', 'admin') DEFAULT 'user',
    membership_level ENUM('free', 'basic', 'premium', 'developer') DEFAULT 'free',
    membership_expire_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

-- 技能表
CREATE TABLE skills (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    skill_id VARCHAR(100) UNIQUE NOT NULL,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    author_id BIGINT NOT NULL,
    category VARCHAR(50),
    tags JSON,
    icon VARCHAR(255),
    version VARCHAR(20),
    download_url VARCHAR(255),
    price DECIMAL(10, 2) DEFAULT 0,
    pricing_model ENUM('free', 'onetime', 'subscription') DEFAULT 'free',
    status ENUM('pending', 'approved', 'rejected', 'disabled') DEFAULT 'pending',
    downloads INT DEFAULT 0,
    rating DECIMAL(2, 1) DEFAULT 5.0,
    rating_count INT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (author_id) REFERENCES users(id)
);

-- 订单表
CREATE TABLE orders (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    skill_id BIGINT,
    membership_level ENUM('basic', 'premium', 'developer'),
    amount DECIMAL(10, 2) NOT NULL,
    currency VARCHAR(3) DEFAULT 'CNY',
    status ENUM('pending', 'paid', 'cancelled', 'refunded') DEFAULT 'pending',
    pay_method VARCHAR(50),
    pay_time TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (skill_id) REFERENCES skills(id)
);

-- 用户技能关系表（已购买/已安装）
CREATE TABLE user_skills (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    skill_id BIGINT NOT NULL,
    relation_type ENUM('purchased', 'installed', 'favorited') NOT NULL,
    expire_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (skill_id) REFERENCES skills(id),
    UNIQUE KEY uk_user_skill_relation (user_id, skill_id, relation_type)
);
```

## 五、API 设计

### 5.1 核心 API

```yaml
# 技能相关
GET   /api/v1/skills              # 获取技能列表
GET   /api/v1/skills/:id          # 获取技能详情
GET   /api/v1/skills/popular      # 热门技能
GET   /api/v1/skills/trending     # Trending
GET   /api/v1/skills/new          # 新技能
GET   /api/v1/skills/search       # 搜索技能
POST  /api/v1/skills              # 上传技能（开发者）
PUT   /api/v1/skills/:id          # 更新技能
DELETE /api/v1/skills/:id         # 删除技能

# 用户相关
POST  /api/v1/auth/register       # 注册
POST  /api/v1/auth/login          # 登录
POST  /api/v1/auth/logout         # 登出
GET   /api/v1/user/profile        # 获取用户信息
PUT   /api/v1/user/profile        # 更新用户信息
GET   /api/v1/user/skills         # 我的技能

# 会员相关
GET   /api/v1/membership/plans    # 会员方案
POST  /api/v1/membership/subscribe # 订阅会员
GET   /api/v1/membership/status   # 会员状态

# 支付相关
POST  /api/v1/payments/create     # 创建订单
POST  /api/v1/payments/notify     # 支付回调
GET   /api/v1/payments/orders     # 订单列表

# 下载相关
GET   /api/v1/download/:version   # 下载软件
GET   /api/v1/download/skills/:id # 下载技能
```

### 5.2 OpenClaw Butler 集成 API

```yaml
# Butler 启动时调用
GET   /api/v1/butler/config       # 获取配置（广告、推荐等）

# 技能市场
GET   /api/v1/butler/skills       # 获取技能列表（带用户权限）
POST  /api/v1/butler/skills/:id/install  # 安装技能
POST  /api/v1/butler/skills/:id/uninstall # 卸载技能

# 用户同步
POST  /api/v1/butler/sync         # 同步用户数据
```
