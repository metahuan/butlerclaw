## 项目概述

基于现有 ButlerClaw 架构，扩展六大核心功能模块，打造 OpenClaw 的**安全与智能管理中心**。

## 一、功能模块设计

### 1. 权限体检（Permission Scanner）

#### 功能描述
一键扫描当前 OpenClaw 的权限配置，识别并提示"过高风险权限"。

#### 检测项
| 检测维度 | 风险等级 | 检测内容 |
|---------|---------|---------|
| Gateway 绑定模式 | 🔴 高危 | 检测是否绑定到 0.0.0.0（公网暴露） |
| 认证配置 | 🔴 高危 | 检测是否缺少 token/password |
| 远程访问配置 | 🟡 中危 | 检测 tailscale/funnel 配置安全性 |
| 文件权限 | 🟡 中危 | 检测 ~/.openclaw 目录权限（是否其他用户可读写） |
| API Key 存储 | 🟡 中危 | 检测是否存在明文存储的 API Key |
| 设备配对 | 🟢 低危 | 检测未授权的设备配对请求 |
| 通道权限 | 🟢 低危 | 检测通道的敏感权限（如发送消息、删除消息） |

#### 输出示例
```
╔══════════════════════════════════════════════════════════════╗
║                    权限体检报告                               ║
╠══════════════════════════════════════════════════════════════╣
║ 🔴 高危风险 (2项)                                            ║
║   ✗ Gateway 绑定到 0.0.0.0:18789，存在公网暴露风险           ║
║   ✗ 未配置 gateway.auth.token，允许匿名访问                 ║
║                                                              ║
║ 🟡 中危风险 (1项)                                            ║
║   ⚠ ~/.openclaw/openclaw.json 权限为 644，建议改为 600      ║
║                                                              ║
║ 🟢 低风险 (1项)                                              ║
║   ℹ 检测到 3 个已配对设备                                    ║
║                                                              ║
║ 综合评分: 45/100 (建议立即修复高危风险)                      ║
╚══════════════════════════════════════════════════════════════╝
```


### 2. 技能查毒（Skill Security Scanner）

#### 功能描述
自动检测已安装的技能包，标记可疑来源，识别潜在安全风险。

#### 检测维度
| 检测项 | 说明 | 风险等级 |
|-------|------|---------|
| 来源验证 | 检查技能是否来自官方仓库 | 🔴 高危 |
| 代码签名 | 验证技能包的数字签名 | 🔴 高危 |
| 敏感权限 | 检测技能是否申请敏感权限 | 🟡 中危 |
| 网络请求 | 检测技能是否包含外部网络请求 | 🟡 中危 |
| 文件操作 | 检测技能是否操作敏感目录 | 🟡 中危 |
| 依赖审计 | 检查技能依赖的安全漏洞 | 🟢 低危 |
| 更新状态 | 检查技能是否为最新版本 | 🟢 低危 |

### 3. 安全加固（Security Hardening）

#### 功能描述
一键关闭不必要的公网访问，完善身份认证，自动应用安全最佳实践。

#### 加固项
| 加固项 | 操作 | 影响 |
|-------|------|------|
| 绑定模式 | 从 0.0.0.0 改为 127.0.0.1 | 仅本地访问 |
| 强制认证 | 启用 token 或 password | 拒绝匿名访问 |
| Tailscale 替代 | 关闭公网，启用 Tailscale | 零信任网络 |
| 文件权限 | 设置 ~/.openclaw 为 700 | 防止其他用户访问 |
| API Key 加密 | 迁移到系统密钥管理 | 更安全存储 |
| 会话超时 | 启用自动会话过期 | 减少风险窗口 |

### 权限体检核心代码

```python
# security/permission_scanner.py
import os
import json
import stat
from typing import List, Dict, Any
from dataclasses import dataclass
from enum import Enum

class RiskLevel(Enum):
    CRITICAL = "critical"
    HIGH = "high"
    MEDIUM = "medium"
    LOW = "low"
    SAFE = "safe"

@dataclass
class PermissionCheck:
    name: str
    level: RiskLevel
    message: str
    details: str
    fix_suggestion: str
    auto_fixable: bool

class PermissionScanner:
    """OpenClaw 权限体检扫描器"""
    
    CONFIG_PATH = os.path.expanduser("~/.openclaw/openclaw.json")
    CONFIG_DIR = os.path.expanduser("~/.openclaw")
    
    RISK_SCORES = {
        RiskLevel.CRITICAL: 0,
        RiskLevel.HIGH: 25,
        RiskLevel.MEDIUM: 50,
        RiskLevel.LOW: 75,
        RiskLevel.SAFE: 100
    }
    
    def scan(self) -> Dict[str, Any]:
        """执行完整权限扫描"""
        checks = []
        
        # 1. 检查 Gateway 绑定模式
        checks.append(self._check_gateway_bind())
        
        # 2. 检查认证配置
        checks.append(self._check_auth_config())
        
        # 3. 检查远程访问配置
        checks.append(self._check_remote_access())
        
        # 4. 检查文件权限
        checks.append(self._check_file_permissions())
        
        # 5. 检查 API Key 存储
        checks.append(self._check_api_key_storage())
        
        # 6. 检查设备配对
        checks.append(self._check_device_pairing())
        
        # 7. 检查通道权限
        checks.append(self._check_channel_permissions())
        
        return self._generate_report(checks)
    
    def _check_gateway_bind(self) -> PermissionCheck:
        """检查 Gateway 绑定模式"""
        config = self._load_config()
        gateway = config.get("gateway", {})
        bind = gateway.get("bind", "auto")
        
        if bind in ["0.0.0.0", "lan", "tailnet"]:
            return PermissionCheck(
                name="Gateway 绑定模式",
                level=RiskLevel.CRITICAL,
                message=f"Gateway 绑定到 {bind}，存在公网暴露风险",
                details="当前配置允许公网访问 Gateway，可能导致未授权访问",
                fix_suggestion="建议将 gateway.bind 改为 'loopback' 或启用强制认证",
                auto_fixable=True
            )
        
        return PermissionCheck(
            name="Gateway 绑定模式",
            level=RiskLevel.SAFE,
            message="Gateway 绑定配置安全",
            details=f"当前绑定模式: {bind}",
            fix_suggestion="",
            auto_fixable=False
        )
    
    def _check_auth_config(self) -> PermissionCheck:
        """检查认证配置"""
        config = self._load_config()
        gateway = config.get("gateway", {})
        
        has_token = bool(gateway.get("auth", {}).get("token"))
        has_password = bool(gateway.get("auth", {}).get("password"))
        
        if not has_token and not has_password:
            return PermissionCheck(
                name="Gateway 认证配置",
                level=RiskLevel.CRITICAL,
                message="未配置认证，允许匿名访问",
                details="gateway.auth.token 和 gateway.auth.password 均未设置",
                fix_suggestion="建议配置 gateway.auth.token 启用强制认证",
                auto_fixable=True
            )
        
        return PermissionCheck(
            name="Gateway 认证配置",
            level=RiskLevel.SAFE,
            message="已配置认证",
            details=f"Token: {'已配置' if has_token else '未配置'}, Password: {'已配置' if has_password else '未配置'}",
            fix_suggestion="",
            auto_fixable=False
        )
    
    def _check_file_permissions(self) -> PermissionCheck:
        """检查配置文件权限"""
        if not os.path.exists(self.CONFIG_PATH):
            return PermissionCheck(
                name="配置文件权限",
                level=RiskLevel.MEDIUM,
                message="配置文件不存在",
                details="",
                fix_suggestion="",
                auto_fixable=False
            )
        
        mode = os.stat(self.CONFIG_PATH).st_mode
        other_read = bool(mode & stat.S_IROTH)
        other_write = bool(mode & stat.S_IWOTH)
        
        if other_read or other_write:
            return PermissionCheck(
                name="配置文件权限",
                level=RiskLevel.MEDIUM,
                message="配置文件权限过宽",
                details=f"其他用户{'可读' if other_read else ''}{'可写' if other_write else ''}",
                fix_suggestion="建议执行 chmod 600 ~/.openclaw/openclaw.json",
                auto_fixable=True
            )
        
        return PermissionCheck(
            name="配置文件权限",
            level=RiskLevel.SAFE,
            message="配置文件权限正确",
            details="仅所有者可读写",
            fix_suggestion="",
            auto_fixable=False
        )
    
    def _check_api_key_storage(self) -> PermissionCheck:
        """检查 API Key 存储方式"""
        config = self._load_config()
        models = config.get("models", {})
        providers = models.get("providers", {})
        
        plaintext_keys = []
        for provider, cfg in providers.items():
            if isinstance(cfg, dict) and cfg.get("apiKey"):
                # 检查是否为明文（长度判断简单方式）
                key = cfg.get("apiKey", "")
                if len(key) > 20 and not key.startswith("${"):
                    plaintext_keys.append(provider)
        
        if plaintext_keys:
            return PermissionCheck(
                name="API Key 存储",
                level=RiskLevel.MEDIUM,
                message=f"{len(plaintext_keys)} 个提供商使用明文存储 API Key",
                details=f"涉及: {', '.join(plaintext_keys)}",
                fix_suggestion="建议迁移到系统密钥管理或使用环境变量",
                auto_fixable=False
            )
        
        return PermissionCheck(
            name="API Key 存储",
            level=RiskLevel.SAFE,
            message="API Key 存储方式安全",
            details="未检测到明文存储",
            fix_suggestion="",
            auto_fixable=False
        )
    
    def _load_config(self) -> Dict:
        """加载配置文件"""
        if not os.path.exists(self.CONFIG_PATH):
            return {}
        try:
            with open(self.CONFIG_PATH, 'r', encoding='utf-8') as f:
                return json.load(f)
        except:
            return {}
    
    def _generate_report(self, checks: List[PermissionCheck]) -> Dict[str, Any]:
        """生成扫描报告"""
        critical = [c for c in checks if c.level == RiskLevel.CRITICAL]
        high = [c for c in checks if c.level == RiskLevel.HIGH]
        medium = [c for c in checks if c.level == RiskLevel.MEDIUM]
        low = [c for c in checks if c.level == RiskLevel.LOW]
        safe = [c for c in checks if c.level == RiskLevel.SAFE]
        
        # 计算综合评分
        if critical:
            score = 0
        elif high:
            score = 25
        elif medium:
            score = 50
        elif low:
            score = 75
        else:
            score = 100
        
        return {
            "score": score,
            "summary": {
                "critical": len(critical),
                "high": len(high),
                "medium": len(medium),
                "low": len(low),
                "safe": len(safe)
            },
            "checks": checks,
            "timestamp": datetime.now().isoformat()
        }
```

```python
        "openai/gpt-4o": {"input": 0.005, "output": 0.015, "currency": "USD"},
        "anthropic/claude-3-5-sonnet": {"input": 0.003, "output": 0.015, "currency": "USD"},
        "google/gemini-2.0-flash": {"input": 0.001, "output": 0.004, "currency": "USD"},
    }
    
    def __init__(self):
        self._init_db()
    
    def _init_db(self):
        """初始化数据库"""
        os.makedirs(os.path.dirname(self.DB_PATH), exist_ok=True)
        conn = sqlite3.connect(self.DB_PATH)
        cursor = conn.cursor()
        
        cursor.execute('''
            CREATE TABLE IF NOT EXISTS cost_entries (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                timestamp TEXT NOT NULL,
                model TEXT NOT NULL,
                provider TEXT NOT NULL,
                input_tokens INTEGER NOT NULL,
                output_tokens INTEGER NOT NULL,
                cost_cny REAL NOT NULL,
                skill_id TEXT,
                session_id TEXT
            )
        ''')
        
        cursor.execute('''
            CREATE INDEX IF NOT EXISTS idx_timestamp ON cost_entries(timestamp)
        ''')
        
        cursor.execute('''
            CREATE INDEX IF NOT EXISTS idx_model ON cost_entries(model)
        ''')
        
        conn.commit()
        conn.close()
    
    def record_usage(self, model: str, input_tokens: int, output_tokens: int, 
                     skill_id: str = None, session_id: str = None):
        """记录一次使用"""
        provider = model.split('/')[0] if '/' in model else 'unknown'
        cost = self._calculate_cost(model, input_tokens, output_tokens)
        
        conn = sqlite3.connect(self.DB_PATH)
        cursor = conn.cursor()
        
        cursor.execute('''
            INSERT INTO cost_entries 
            (timestamp, model, provider, input_tokens, output_tokens, cost_cny, skill_id, session_id)
            VALUES (?, ?, ?, ?, ?, ?, ?, ?)
        ''', (
            datetime.now().isoformat(),
            model,
            provider,
            input_tokens,
            output_tokens,
            cost,
            skill_id,
            session_id
        ))
        
        conn.commit()
        conn.close()
    
    def _calculate_cost(self, model: str, input_tokens: int, output_tokens: int) -> float:
        """计算成本 (转换为 CNY)"""
        pricing = self.PRICING.get(model, {"input": 0.01, "output": 0.01, "currency": "CNY"})
        
        input_cost = (input_tokens / 1000) * pricing["input"]
        output_cost = (output_tokens / 1000) * pricing["output"]
        total = input_cost + output_cost
        
        # 如果是 USD，按 7.2 汇率转换
        if pricing["currency"] == "USD":
            total *= 7.2
        
        return round(total, 4)
    
    def get_daily_stats(self, days: int = 7) -> List[Dict]:
        """获取每日统计"""
        conn = sqlite3.connect(self.DB_PATH)
        cursor = conn.cursor()
        
        since = (datetime.now() - timedelta(days=days)).isoformat()
        
        cursor.execute('''
            SELECT 
                date(timestamp) as day,
                SUM(input_tokens) as input_tokens,
                SUM(output_tokens) as output_tokens,
                SUM(cost_cny) as total_cost,
                COUNT(*) as request_count
            FROM cost_entries
            WHERE timestamp > ?
            GROUP BY date(timestamp)
            ORDER BY day DESC
        ''', (since,))
        
        results = []
        for row in cursor.fetchall():
            results.append({
                "date": row[0],
                "input_tokens": row[1],
                "output_tokens": row[2],
                "cost_cny": round(row[3], 2),
                "request_count": row[4]
            })
        
        conn.close()
        return results
    
    def get_model_breakdown(self, days: int = 30) -> List[Dict]:
        """获取模型消费占比"""
        conn = sqlite3.connect(self.DB_PATH)
        cursor = conn.cursor()
        
        since = (datetime.now() - timedelta(days=days)).isoformat()
        
        cursor.execute('''
            SELECT 
                model,
                SUM(cost_cny) as total_cost,
                SUM(input_tokens + output_tokens) as total_tokens
            FROM cost_entries
            WHERE timestamp > ?
            GROUP BY model
            ORDER BY total_cost DESC
        ''', (since,))
        
        results = []
        total = 0
        rows = cursor.fetchall()
        for row in rows:
            total += row[1]
        
        for row in rows:
            results.append({
                "model": row[0],
                "cost_cny": round(row[1], 2),
                "tokens": row[2],
                "percentage": round(row[1] / total * 100, 1) if total > 0 else 0
            })
        
        conn.close()
        return results
    
    def predict_monthly_cost(self) -> Dict:
        """预测本月消费"""
        daily_stats = self.get_daily_stats(days=7)
        
        if not daily_stats:
            return {"predicted": 0, "daily_average": 0}
        
        # 计算7天平均日消费
        total_cost = sum(d["cost_cny"] for d in daily_stats)
        daily_avg = total_cost / len(daily_stats)
        
        # 获取本月已消费
        today = datetime.now()
        first_day = today.replace(day=1)
        
        conn = sqlite3.connect(self.DB_PATH)
        cursor = conn.cursor()
        
        cursor.execute('''
            SELECT SUM(cost_cny) as total
            FROM cost_entries
            WHERE timestamp >= ?
        ''', (first_day.isoformat(),))
        
        row = cursor.fetchone()
        month_so_far = row[0] or 0
        
        conn.close()
        
        # 预测本月总消费
        days_remaining = 30 - today.day
        predicted = month_so_far + (daily_avg * days_remaining)
        
        return {
            "predicted": round(predicted, 2),
            "daily_average": round(daily_avg, 2),
            "month_so_far": round(month_so_far, 2)
        }
```
